# AI Healthcare Security Lab

An intentionally vulnerable AI-powered healthcare platform designed for:

- LLM security testing
- Multi-agent security research
- Prompt injection testing
- Agent hijacking research
- Memory poisoning testing
- Tool abuse testing
- Data leakage testing
- MITRE ATLAS research
- OWASP LLM security testing

## Project Status

Currently under active development.

## Current Features

- FastAPI backend
- PostgreSQL database
- Patient data API
- SQLAlchemy database integration

## Planned Features

- Multi-agent architecture
- AI healthcare chatbot
- Medical records agent
- Triage agent
- Diagnosis support agent
- Prescription agent
- Scheduling agent
- Long-term memory
- Vector database
- File upload and parsing
- Email ingestion
- Plugin system
- Tool execution layer
- Security testing scenarios
